# Password Strength Checker

This is a Java console application that evaluates password strength based on common security rules.

## Features
- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers and special characters
- Rates password as Weak, Medium, or Strong
- Provides feedback to improve password strength
- Includes menu system and error handling

## Skills Demonstrated
- Java programming
- Input validation
- Conditional logic (if/else)
- Loops and control flow
- Regular expressions
- Problem solving

## How to Run
1. Open in Eclipse or any Java IDE
2. Run the `PasswordStrengthChecker` class
3. Follow the menu options
